2.SSL证书错误

因为1中使用http，80端口访问的，也就是从nginx container到wordpress container中访问是通过HTTP访问的，由于wordpress container中没有配置SSL证书，目前也只能这样了。正因为这样，wordpress中检测不到https访问，所以资源链接一律返回HTTP，到浏览器一片红。
